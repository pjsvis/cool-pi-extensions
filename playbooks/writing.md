# Playbook: writing/

## Purpose

Long-form essays and technical writing. Deeper explorations of ideas that don't fit in briefs, debriefs, or canon pages.

## Directory structure

```
writing/
├── essay-slug.md
└── ...
```

## Conventions

- Markdown format
- Filename: kebab-case descriptive slug
- Self-contained — each essay stands alone
- Include date in the header or first line
- Link to related briefs, debriefs, and canon pages

## Difference from other silos

| Silo | Purpose |
|---|---|
| `briefs/` | Specification — what to build |
| `debriefs/` | Reflection — what we learned |
| `decisions/` | Rationale — why we chose X |
| `canon/` | Reference — permanent definitions |
| `writing/` | Exploration — deeper thinking |
| `playbooks/` | Instructions — how to do things |

Writing is for ideas that need more space than a brief but aren't formal enough for canon. Think: philosophical explorations, technical deep-dives, project postmortems that go beyond the debrief format.
